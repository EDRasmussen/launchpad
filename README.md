# LaunchPad

Personal homepage doubling as a reference project for various functionality.

Built with

- TanStack Start
- TanStack Query
- TanStack Forms
- Drizzle ORM
- ShadCN UI

## Development

```bash
bun install
bunx drizzle-kit migrate
bun run dev
```

## Seed the database with:

```bash
bun run seed
```
